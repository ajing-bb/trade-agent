# Style Bible Template

Use this template when the project needs a stable show-level visual canon across episodes.

Purpose:
- separate show-level style from character-level or scene-level prompts
- reduce style drift in sequels
- make “what this show looks like” explicit in the repository

## Output Shape

- `Style Summary`
- `Line and Render Rules`
- `Color and Light Rules`
- `World Design Rules`
- `Camera and Composition Rules`
- `Failure Boundaries`

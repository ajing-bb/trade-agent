# Character Bible Template

Use this template when the user has recurring characters across multiple shots or episodes.

## How To Use

- Fill one row per recurring character.
- Keep “must not drift” fields concrete.
- Add angle packs only for shots the project actually needs.
- If the project has extras, define reusable background-character archetypes instead of unique full bibles for each extra.

## Minimum Fields

At minimum, always fill:
- `Visual Core`
- `Face Anchors`
- `Costume Core`
- `Color Anchors`
- `Required Angles`
- `Must Not Drift`

## Must Not Drift Examples

Good:
- “额前碎发轮廓、细长眼型、学院制服领口结构”
- “红风衣长度、面具形状、肩部比例”
- “发簪位置、道袍层次、手持武器轮廓”

Bad:
- “气质”
- “好看”
- “像原来一样”

## Angle Pack Rule

Only request extra angle packs when a shot needs them:
- dialogue-heavy: front, 3/4, side
- action-heavy: front, side, low angle, high angle
- reverse-shot scene: over-shoulder-friendly side packs
- reveal shots: low angle or silhouette versions

# Scene Master Template

Use this template for every recurring location before writing many prompts.

## How To Use

- Fill one row per important recurring scene.
- A `Master Shot` should describe the widest useful view of that location.
- Record spatial anchors so later angle changes still feel like the same place.
- If a scene has both day and night versions, treat them as separate rows unless the layout is identical and only lighting changes.

## Spatial Anchor Rule

Every recurring scene should have at least 4 concrete anchors:
- architecture
- prop/device
- pathway or floor pattern
- skyline / background shape

If the scene does not have these anchors, later shots will drift.

## Camera Seed Rule

List angle seeds only when needed:
- `front wide`
- `side wide`
- `reverse / over-shoulder`
- `low-angle hero`
- `high-angle tactical`

Do not create every angle by default.

## Crowd Policy

When the scene includes many people, decide early:
- whether extras are soft-background only
- whether a few named extras need stable identities
- whether the crowd should be split into foreground and background layers

## Must Not Drift Examples

Good:
- “抽卡池位置永远在操场中央，背后能看到学院主体建筑”
- “食堂窗口、立柱、长桌排布保持一致”
- “公共抽卡池夜景必须有城市天际线和红色异象映射”

Bad:
- “差不多像同一个地方”

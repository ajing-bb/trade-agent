# 第一集资产规划（80秒版）

## 目标

- 第一集时长控制在 `70-90秒`
- 当前执行版本按 `80秒左右` 规划
- 本集只覆盖 `开场世界钩子 + 孟江第十抽通天教主异象 + 当场失效反转`
- 食堂线与哪吒夜线不进入第一集首轮资产包

## 秒数分配

| 时段 | 剧情内容 | 需要调用的资产 |
| --- | --- | --- |
| `0-10s` | 世界观开场，操场、抽卡池、西方神魔体系快速建立 | 神风学院操场、抽卡池、西方神魔雕像组 |
| `10-28s` | 孟江上台，学生群嘲，主角压抑蓄势 | 孟江、操场场景、学生群演模板 |
| `28-52s` | 天空变色、金色异象、校长办公室反应 | 金色天幕、七彩光柱、校长办公室、司徒昆仑、司徒青青 |
| `52-68s` | 通天教主卡出场、老教授震惊、孟江贴卡 | 通天教主卡、老教授、抽卡池、孟江 |
| `68-80s` | 异象消失、全场嘲笑、孟江失落收尾 | 金色异象消退版、操场、孟江、学生群演 |

## 出场角色

| 角色 | 第一集用途 | 建议资产级别 |
| --- | --- | --- |
| `孟江` | 绝对主角，几乎贯穿全集 | 必须做完整包：`face draft + full body + 3/4 + side + low-angle hero + 情绪近景` |
| `司徒昆仑` | 用来放大“这次异象不一般” | 做中配包即可：`半身主视角 + 3/4 + 办公室反应` |
| `司徒青青` | 给校长反应戏做对手和解释口 | 做轻量包：`半身 + 3/4 + 侧脸` |
| `老教授` | 卡牌 reveal 前后的权威惊诧反应 | 做轻量包：`半身惊讶表情 + 近景` |
| `学生群演` | 群嘲、围观、收尾嘲笑 | 不单独逐人定稿，先做 `3个学生模板 + 1个惊讶反应模板` 循环复用 |

## 需要的场景

| 场景 | 是否第一集必须 | 用法 |
| --- | --- | --- |
| `LOC_KAMIKAZE_ACADEMY_GROUNDS_DAY` 神风学院操场 | 必须 | 第一集主场景，开场、抽卡、异象、失效全在这里 |
| `LOC_PRINCIPAL_OFFICE_DAY` 校长办公室 | 建议保留 | 用一个短 cut-in 就能把异象规格抬上去 |
| `LOC_ACADEMY_CAFETERIA_DAY` 食堂 | 第一集不用 | 留给第二集 |
| `LOC_PUBLIC_DRAW_POOL_NIGHT` 夜间公共抽卡池 | 第一集不用 | 留给第三集 |

| VFX | 是否第一集必须 | 说明 |

| `VFX_GOLDEN_SKY_ANOMALY` 金色天幕异象 | 必须 | 第一集最大的视觉卖点 |

| `VFX_RED_SKY_ANOMALY` / `VFX_RED_LIGHT_PILLAR` / `VFX_NEZHA_*` | 第一集不用 | 全部留到第三集 |

## 第一集最小可执行资产包

- `角色`：孟江、司徒昆仑、司徒青青、老教授、学生模板 3-4 个
- `场景`：神风学院操场日景 master、校长办公室日景 master
- `道具`：抽卡池、西方神魔雕像组、通天教主卡
- `VFX`：金色天幕、七彩光柱、异象消退版

## 推荐制作顺序

1. 先锁 `孟江`
2. 再锁 `神风学院操场日景 + 抽卡池`
3. 再做 `通天教主卡 + 金色天幕 + 七彩光柱`
4. 再补 `司徒昆仑 / 司徒青青 / 老教授`
5. 最后做学生群演模板和收尾嘲笑镜头

## 第一集先不要做

- 食堂整套资产
- 林倩倩、张贺、跟班角色包
- 夜间公共抽卡池
- 哪吒卡、哪吒法器组、哪吒虚像
- 赤红天幕整套 VFX

## 备注

- 第一集的目标不是把世界一次讲完，而是先把 `孟江 + 超规格异象 + 当场失效反转` 这个钩子打稳
- 视觉策略上优先保证 `孟江一致性` 和 `操场空间连续性`
- 强特效镜头优先走 `scene plate + effect overlay`，不要让 VFX 直接吞掉角色脸

# EP001 Continuity Plan

## 高优先级锁定项

1. 孟江、张贺、林倩倩、司徒昆仑的已落盘角色包作为当前唯一 canonical 角色基线。
2. 司徒青青保留在角色 bible 中，但当前仍需落盘正式资产。
3. 学院白天线和夜间公共抽卡池线必须分别锁两套 scene master。
4. 通天教主线和哪吒线的 VFX 视觉语言必须分离。

## Shot Build Policy

| Shot ID | Build Method | Why |
| --- | --- | --- |
| `EP001-S00-SH01` | `master-shot crop + montage` | 世界观建立，重 geography |
| `EP001-S01-SH01` | `still first` | 孟江与操场位置必须稳定 |
| `EP001-S01-SH02` | `first/last frame` | 白天大异象最容易漂 |
| `EP001-S01-SH03` | `image merge` | 通天教主卡面与失败反应需要稳定 |
| `EP001-S02-SH01` | `still first + reverse coverage` | 食堂戏核心是人物关系与反打 |
| `EP001-S03-SH01` | `still first + first/last frame` | 夜景抽卡启动需要稳住场地与光效 |
| `EP001-S04-SH01` | `scene seed + montage` | 多地反应只需要共享异象天色 |
| `EP001-S05-SH01` | `layered video + regional composite` | 哪吒法相与法器高风险 |

## Continuity Risks

| 风险 | 说明 | 对策 |
| --- | --- | --- |
| 制服体系漂移 | 新角色可能脱离黑金学院体系 | 服装统一全部走 Banana Pro |
| 张贺目录异常 | 当前目录名有前置空格 | manifest 中保留真实路径，后续统一前先不改 |
| 司徒青青未落盘 | 角色存在但仓库还没图 | 先记录 planned，再补落盘 |

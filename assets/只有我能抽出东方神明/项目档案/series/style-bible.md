# Style Bible

## Style Summary

| Field | Value |
| --- | --- |
| Project Style Name | 国风玄幻校园 2.5D 漫剧 |
| Core Format | 2.5D manhua |
| Overall Density | medium |
| Realism Level | semi-flat |
| Reference Warning | 不要漂成乙游立绘、韩系写真、过萌甜妹日漫、过写实厚涂 |

## Line and Render Rules

| Category | Rule |
| --- | --- |
| Line Quality | clean comic linework，主轮廓清楚，不走厚重写实线 |
| Shadow Depth | 2-step 到轻 2.5D，允许一定体积感，但不能走重写实 |
| Skin Finish | matte skin，轻柔过渡，不做油亮写真皮肤 |
| Hair Finish | 以成组发束为主，保留少量散发，避免过度丝发写实 |
| Fabric Finish | 中等褶皱，优先结构可读性而不是材质炫技 |

## Color and Light Rules

| Category | Rule |
| --- | --- |
| Day Palette | 冷灰校园基底，局部金色异象覆盖 |
| Night Palette | 冷蓝黑夜景基底，赤红异象压顶 |
| Gold / Fire / Energy Palette | 通天教主线偏古朴金彩祥光，哪吒线偏赤金火焰 |
| Academy Base Palette | 黑、白、暗金为校服与制度感主色 |
| Contrast Policy | medium contrast，重要 VFX 可临时提高 |
| Highlight Policy | restrained highlights，避免 glossy 乙游表面质感 |

## World Design Rules

| Category | Rule |
| --- | --- |
| School Uniform System | 同校统一黑金体系，男主标准版、张贺精英版、女生同系统变体 |
| Architecture Language | 西式魔法学院建筑 + 西方神魔雕像 |
| Prop Language | 抽卡池、卡牌、法器都要有清晰可读轮廓 |
| VFX Split Rules | 通天教主线与哪吒线不可混色混材质 |
| East vs West Visual Contrast | 西方体系走学院雕像与制度感，东方体系走神性异象与法器辨识 |

## Camera and Composition Rules

| Category | Rule |
| --- | --- |
| Default Framing | 角色建立优先中景和全身设定图 |
| Dialogue Coverage | 反打稳定，先保证角色关系，不追复杂运动 |
| Reveal Shots | 允许低角度英雄镜头，但要先有稳定资产 |
| VFX Hero Shots | 先分层锁静帧，再做动态 |
| Crowd Policy | 前景 4-6 个可辨识模板，中后景软化处理 |

## Failure Boundaries

- 不要漂成 glossy 乙游男主 / 女主立绘
- 不要漂成韩系写真脸
- 不要漂成过萌日漫甜妹
- 不要漂成电影级厚涂写实
- 不要让通天教主线和哪吒线使用同一种异象语言

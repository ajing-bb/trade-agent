## 世界观硬规则

1. 所有人年满十八即可抽卡。
2. 抽到的卡片会赋予对应神魔能力。
3. 这个世界的默认神魔体系是西方神魔。
4. 孟江是穿越者，是当前已知唯一可以召唤东方神明的人。
5. 抽卡异象强度和神明/卡片层级正相关，但异象不等于必然成功激活。

## 审美与制作规则

1. 先锁角色、再锁场景、再锁道具和 VFX，最后才做视频。
2. 学院体系统一为黑色主制服、暗金结构线、白色内领的世界语言。
3. 孟江是标准男生制服线，张贺是同校精英版变体。
4. 林倩倩与司徒青青属于同校女生制服体系，但气质线分开：
   - 林倩倩：校花 / 前女友 / 轻蔑感
   - 司徒青青：校方千金 / 端正 / 理性
5. `Midjourney` 只做底图，`Banana Pro` 统一做服装统一、方向修正、单格提取与回拼。

## Canonical 使用规则

1. 本目录与 `episodes/` 中已确认的档案文件，构成本项目的 canonical memory。
2. 后续续集、资产补画、Seedance prompt、角色返修，默认先读 canonical 档案，再参考聊天记录。
3. 如果聊天结论与仓库档案冲突，以仓库中最新确认版本为准。
4. 临时探索图、未落盘图片、未写回档案的 prompt 结论，不视为 canonical。

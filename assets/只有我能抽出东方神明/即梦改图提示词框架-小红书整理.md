# 即梦改图提示词框架

基于 2026-03-19 通过小红书抓取的“即梦改图 / 局部重绘 / 垫图改图”内容整理，目标是给 `只有我能抽出东方神明` 项目沉淀一套可直接复用的中文提示词框架。

## 这次整理出来的核心结论

1. 即梦改图最关键的不是堆关键词，而是把这三件事写清楚：
   - `改哪里`
   - `保留什么不动`
   - `改成什么`
2. 小红书上的高频有效经验都在强调：
   - 局部重绘前先精确圈区域
   - 一次只改一个核心目标
   - 提示词尽量短、准、局部，不要写成长篇生图 prompt
3. 如果是垫图或局部重绘，最稳的写法是：
   - `只修改局部 + 保留主体 + 指定目标变化 + 补充材质/光影`
4. 改图翻车通常不是“词不够多”，而是：
   - 改动范围没锁清
   - 保留项没写
   - 一次改太多

## 小红书里能直接确认的实战信号

以下内容是基于实际抓到的笔记标题和正文摘要整理，不是凭空猜的。

### 1. 局部重绘适合 70-80 字左右的精准描述

来源：

- `6864cbd0000000001203dc8b`《即梦100个局部重绘指令！速存！》

笔记明确强调：

- 面部优化、服装材质、光影氛围、饰品细节都适合直接用局部重绘处理
- 70-80 字左右的精准描述就足够，不需要很长

### 2. 垫图改图可以“一句话搞定”

来源：

- `6860b00a000000001d00f380`《改图只用一句话！16种进阶垫图玩法提示词》

笔记明确给出了改图适用范围：

- 换背景修图
- 换装
- 模特图
- 氛围图
- 保持 IP 一致性
- 角色替换
- 照片转插画
- 2D 转 3D
- 老照片翻新

这说明即梦改图天然适合“基于现有图做定向改动”，而不是每次重头生。

### 3. 局部重绘前要先把区域圈准

来源：

- `691fb4ae000000001e026a2d`《既梦4.0局部重绘🔥超全电商案列精准出图》

笔记里最有价值的点是：

- 先把要调整的区域圈出来
- 参考图的提示词可以反推后再改写成自己的提示词

这对你后面做角色修脸、修制服、修头发非常重要。

### 4. 人物局部重绘最怕整图被重画

来源：

- `694fd4880000000022030167`《即梦AI人物局部重绘🔥20个指令赶紧收藏》

笔记里说得很直接：

- 很多人只是想换一个衣服、动作、背景
- 结果整个画面被重绘，风格跑偏

所以人物改图 prompt 一定要显式写“保留项”。

## 适合即梦改图的最稳框架

### 极简版

`只修改[区域]，保留[主体/脸/发型/服装主体/构图/画风]不变，将[区域]改为[目标效果]。`

### 标准版

`只修改[具体区域]，保留[人物身份/脸型/发型/服装主体/动作/构图/原有画风]不变，将[该区域]改为[目标内容]，[补充材质/光影/颜色/细节]，整体保持[原图风格]。`

### 强约束版

`只重绘[区域]，不要改动[列出必须保留的内容]，把[区域]调整为[目标变化]，要求[材质/颜色/边缘/光影/细节]，保持原图线稿、配色关系、人物比例和镜头构图不变。`

## 我建议你以后固定用的 4 段式写法

### 1. 修改范围

先锁区域，避免整图被带跑。

常用写法：

- `只修改人物脸部区域`
- `只修改上半身制服区域`
- `只修改背景区域`
- `只修改头发和刘海区域`
- `只修改右手和手中道具区域`

### 2. 保留项

这一段是即梦改图最容易漏掉、但最关键的部分。

常用写法：

- `保留人物身份、脸型、发型轮廓不变`
- `保留原图构图、姿势和镜头不变`
- `保留原图画风和线稿质感不变`
- `保留服装主体结构不变`
- `保留人物年龄感和身材比例不变`

### 3. 目标变化

只写一个核心目标，不要一条 prompt 同时换脸、换衣服、换背景、换动作。

常用写法：

- `把制服领口改为更利落的高领结构`
- `把背景改成浅灰色纯净背景`
- `把头发改为更整洁的深黑长直发`
- `把嘴角调整为更克制的平直线条`
- `把耳饰改成小型银色耳钉`

### 4. 补充细节

这一步只补会影响最终质感的东西。

常用写法：

- `材质保持哑光布料`
- `金属细节少量克制`
- `光影保持柔和，不要强逆光`
- `整体保持半平涂、清晰线稿`
- `颜色保持深蓝黑、白、低饱和银灰`

## 最推荐的 6 类即梦改图模板

### 1. 修脸

```text
只修改人物脸部区域，保留人物身份、发型轮廓、原图画风和构图不变，把眼型调整为更锐一点的杏眼，嘴角微微收紧，唇线更平直，下巴更利落，五官精致但不甜，整体保持清冷观察型气质。
```

### 2. 修头发

```text
只修改头发和刘海区域，保留人物脸型、表情、服装和构图不变，把头发调整为更整洁的深黑长直发，发丝顺直清晰，刘海利落，不要蓬松甜妹卷度，整体保持原图画风。
```

### 3. 修制服

```text
只修改上半身制服区域，保留人物脸、发型、姿势和镜头不变，把制服改成更利落的深色高领学院制服，云纹只出现在饰边和徽记，布料哑光偏挺，少量银色细节，保持精英学院感，不要改成礼服或古装。
```

### 4. 修背景

```text
只修改背景区域，保留人物本身、动作、光影关系和画风不变，把背景改成干净的浅灰色纯背景，弱化杂乱元素，让人物更突出。
```

### 5. 换局部动作或手部

```text
只修改右手和手部姿态区域，保留人物脸、身体比例、服装和整体构图不变，把右手动作调整为自然握持姿势，手指结构清晰自然，不要畸形，不要改变人物整体站姿。
```

### 6. 修风格跑偏

```text
只调整整体局部细节，保留人物身份、构图和主体内容不变，把画面统一回清晰漫画线稿、2D 国漫静帧、半平涂、克制边缘光、哑光布料的方向，不要韩系写真脸，不要乙游高光，不要高饱和萌系。
```


```text
只修改人物局部细节，保留孟江的脸型、黑色短发、身材比例和构图不变，把整体细节统一回清晰漫画线稿、2D 国漫静帧、半平涂、克制边缘光、哑光学院制服的方向，强化隐忍少年感，不要韩系写真脸，不要强写实电影质感。
```

## 使用顺序建议

1. 先圈区域。
2. 先写“只修改哪里”。
3. 再写“保留什么不变”。
4. 再写“改成什么”。
5. 最后只补 1-2 个材质或光影词。

## 不要这样写

- 不要把整张生图 prompt 直接塞给改图。
- 不要一次改 4 件事。
- 不要只写“更好看”“更高级”“更精致”这种空词。
- 不要不写保留项。
- 不要不圈区域就直接改人物。
